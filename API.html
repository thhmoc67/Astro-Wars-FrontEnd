collision :-
User Shuttle will collide with opponent's shuttle then collision function will be executed.
inView :-
If opponent shuttle is at deg(90) straight in user's shuttle then inView function will executed.
bulletHit :-
If user's shuttle is hitted by opponent shuttle then bullet hit will executed.
wallHit :-
If user's shuttle hitted by wall then wallHit will executed.
